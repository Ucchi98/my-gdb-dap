# my-gdb-dap
VSCode(OSS版)でOpenOCDとGDB（要DAP対応版）を使用してデバッグを行えるようにする拡張機能

GhostBSDにインストールしたvscode-ossを使用して、
ESP-IDFのツールチェーンのopenocdとgdbでデバッグを行うために作成しました。

## ビルド方法
% git clone https://github.com/Ucchi98/my-gdb-dap.git

% cd my-gdb-dap

% make

## インストール
生成されたmy-gdb-dap.vsixファイルを、vscodeの拡張機能としてインストールする。

## 使い方
sampleフォルダにあるlaunch.jsonを参考に、必要な設定を行う。
